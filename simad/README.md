# SIMAD - Sistema Inteligente de Marketing de Alto Desempenho

Dashboard completo para gestão de marketing da Alice Werlang com dados reais.

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador
2. Navegue pelo menu lateral para acessar cada seção
3. Importe dados reais das plataformas na seção "Importar Dados"

## Funcionalidades

### Dashboard
- KPIs em tempo real (Faturamento, Leads, CAC, ROAS, Conversão, CTR)
- Gráficos de Receita vs Investimento
- Performance por Canal
- Meta do Dia com barra de progresso
- Ações Rápidas
- Preview do Conteúdo do Dia

### Geração de Conteúdo
- **Instagram:** Posts, Stories e Reels com copies prontas
- **TikTok:** Vídeos com roteiro e hooks virais
- **LinkedIn:** Artigos e posts de texto

### Calendário Editorial
- Visualização mensal completa
- Cores por plataforma
- Geração automática de conteúdo

### Relatórios
- Relatório Diário com todas as métricas
- Análise do que funcionou e não funcionou
- Comparativos e alertas
- Ações prioritárias para o dia seguinte

### Tráfego Pago
- Estrutura TOFU/MOFu/BOFu
- Copies de anúncios (3 variações)
- Segmentação detalhada
- Métricas de campanha

### SEO
- Score de SEO Local
- Palavras-chave alvo
- Calendário de conteúdo SEO
- Briefings de artigos

### Análise de Concorrência
- Benchmark de concorrentes (Tortlas, Pamela Gourmet, Alê Pessoa)
- Análise SWOT completa
- Gaps e oportunidades

### Email Marketing
- Sequências automatizadas
- Welcome Series
- Abandono de Carrinho
- Pós-Compra
- Métricas de email

### Importação de Dados
- **Meta Ads:** CPC, CTR, ROAS, CPL
- **Google Analytics:** Sessões, Usuários, Conversões
- **Instagram:** Seguidores, Alcance, Engajamento
- **Email Marketing:** Open Rate, CTR, Receita
- **Vendas:** Receita, Pedidos, Ticket Médio
- **WhatsApp:** Leads, Conversas, Vendas

### Configuração de APIs
- Meta Business Suite
- Google Analytics
- Google Ads
- TikTok Ads
- RD Station
- HubSpot CRM

## Tecnologias

- HTML5
- CSS3 (Dark Theme)
- JavaScript Vanilla
- Chart.js (Gráficos)
- Font Awesome (Ícones)

## Estrutura

```
simad/
├── index.html      # Página principal
├── css/
│   └── style.css   # Estilos
├── js/
│   ├── app.js      # Lógica principal da aplicação
│   ├── data.js     # Dados reais da empresa
│   └── import.js   # Sistema de importação de dados
├── assets/
│   ├── logo.png    # Logo da empresa
│   └── avatar.png  # Avatar do usuário
└── README.md       # Documentação
```

## Personalização

Para atualizar os dados da empresa, edite o arquivo `js/app.js` e modifique o objeto `state.company`.
