# Guia Rápido - SIMAD

## Início Rápido

### 1. Abrir o Sistema
```
C:\Users\SMEINFO_GUILHERME\Documents\simad\index.html
```
Dê um duplo-clique no arquivo `index.html` para abrir no navegador.

### 2. Importar Dados Reais

#### Meta Ads (Facebook/Instagram)
1. Acesse o Meta Business Suite
2. Vá em **Relatórios** > **Exportar dados**
3. Selecione o período desejado
4. Copie os dados e cole na seção **Importar Dados** do SIMAD

#### Google Analytics
1. Acesse o Google Analytics
2. Vá em **Relatórios** > **Aquisição** > **Todo o tráfego**
3. Anote: Sessões, Usuários, Taxa de Rejeição
4. Importe no SIMAD

#### Instagram
1. Acesse seu perfil no Instagram
2. Vá em **Insights** (Análises)
3. Anote: Seguidores, Alcance, Engajamento
4. Importe no SIMAD

#### WhatsApp Business
1. Acesse o WhatsApp Business
2. Vá em **Ferramentas de negócios** > **Métricas**
3. Anote: Conversas, Pedidos, Receita
4. Importe no SIMAD

### 3. Acompanhar Métricas

#### Dashboard Principal
- **KPIs:** Faturamento, Leads, CAC, ROAS, Conversão, CTR
- **Gráficos:** Receita vs Investimento, Performance por Canal
- **Meta do Dia:** Acompanhe o progresso diário

#### Calendário Editorial
- Visualize todo o conteúdo planejado
- Filtre por plataforma
- Gere conteúdo automaticamente

#### Relatórios
- **Diário:** Métricas completas do dia
- **Semanal:** Resumo da semana
- **Mensal:** Análise completa do mês

### 4. Configurar APIs (Opcional)

Para atualização automática dos dados:

1. Vá em **Importar Dados** > **Configurações de API**
2. Selecione a plataforma
3. Insira o Access Token
4. Teste a conexão
5. Salve a configuração

## Solução de Problemas

### Dados não aparecem
1. Verifique se os dados foram importados
2. Atualize a página (F5)
3. Verifique o console do navegador (F12)

### Gráficos não carregam
1. Verifique sua conexão com a internet
2. O Chart.js precisa de conexão para carregar
3. Recarregue a página

### API não conecta
1. Verifique se o token está correto
2. Confirme se a API está habilitada
3. Teste a conexão novamente
